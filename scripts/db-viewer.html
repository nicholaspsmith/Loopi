MemoryLoop Database Viewer

Quick Commands

View PostgreSQL Data:

npx tsx scripts/view-postgres.ts messages 20
npx tsx scripts/view-postgres.ts users
npx tsx scripts/view-postgres.ts conversations
npx tsx scripts/view-postgres.ts api_keys

View LanceDB Data:

npx tsx scripts/view-lancedb.ts flashcards 20
npx tsx scripts/view-lancedb.ts messages
npx tsx scripts/view-lancedb.ts review_logs

PostgreSQL (psql):

psql $DATABASE_URL

# Common queries:
\dt                           # List tables
\d messages                   # Describe table
SELECT * FROM messages LIMIT 10;
SELECT COUNT(*) FROM flashcards;

Recommendation: Use the npx tsx scripts/view-*.ts commands above for the easiest viewing experience.
